# 📊 Data Analysis Project using SQL and Excel

## 📌 Project Description
This project focuses on data analysis using SQL and Excel. SQL is used for data extraction and filtering, while Excel is used for data cleaning, analysis, and dashboard creation.

## 🛠 Tools Used
- SQL (for data extraction and querying)
- Microsoft Excel (for data cleaning and dashboard creation)

## 📁 Files Included
- SQL file (queries used for analysis)
- Excel file (dashboard and analysis)
- Dashboard images (visual representation)

## 📈 Key Insights
- Data analysis helped identify important trends and patterns  
- SQL queries were used to extract required and filtered data  
- Excel dashboard provides clear visual representation of data  
- Comparison of different categories/fields using charts  
- Key metrics and performance insights were identified  

## 🧠 Solution / Approach
- Raw data was imported into Excel  
- Data cleaning and formatting was performed  
- SQL queries were used for data extraction and filtering  
- Processed data was analyzed in Excel using pivot tables  
- Dashboard was created using charts and visuals
